시작 준비

디스코드 앱 설치하기

PC와 모바일 모두 설치하면 언제 어디서든 소통할 수 있습니다. PC 앱을 먼저 설치하는 것을 권장합니다.

디스코드 설치 과정 영상

순서대로 따라하세요

  1. 1

    디스코드 다운로드 페이지에 접속합니다.

    https://discord.com/download

  2. 2

    화면에 보이는 큰 파란색 다운로드 버튼을 클릭합니다.

    사용 중인 운영체제(Windows / Mac)에 맞는 버전이 자동으로 선택됩니다.

  3. 3

    다운로드된 설치 파일을 실행합니다.

    Windows: DiscordSetup.exe / Mac: Discord.dmg

  4. 4

    설치가 완료되면 디스코드 앱이 자동으로 실행됩니다.

💡
모바일에서도 사용하려면 앱스토어(iOS) 또는 플레이스토어(Android)에서 "Discord"를 검색하여 설치하세요.

✓
PC에 디스코드 앱을 설치했다
✓
앱이 정상적으로 실행된다

회원가입하기

디스코드 계정을 만듭니다. 이미 계정이 있다면 이 단계를 건너뛰세요.

이메일 입력부터 인증까지 전체 과정

순서대로 따라하세요

  1. 1

    디스코드 앱을 실행하고, "가입" 버튼을 클릭합니다.

  2. 2

    이메일 주소를 입력합니다.

    📌
    PM인 경우: 반드시 회사 이메일 @sandboxnetwork.net으로 가입하세요.
  3. 3

    표시 이름을 입력합니다. 나중에 바꿀 수 있으니 편하게 입력하세요.

  4. 4

    비밀번호를 설정합니다.

  5. 5

    생년월일을 입력하고 "계속"을 클릭합니다.

  6. 6

    이메일로 온 인증 링크를 클릭해서 가입을 완료합니다.

프로필 설정하기

이름을 실명으로 설정해서 팀원들이 바로 알아볼 수 있게 합니다.

사용자 설정에서 프로필 이름 변경하는 과정

순서대로 따라하세요

  1. 1

    디스코드 화면 왼쪽 하단에서 자기 이름 옆의 톱니바퀴 아이콘(⚙️)을 클릭합니다.

  2. 2

    왼쪽 메뉴에서 "프로필 편집"을 클릭합니다.

  3. 3

    "별명"을 클릭하여 수정합니다.

  4. 4

    이름을 샌드박스 실명으로 변경합니다.

    예시

    샌드박스 홍길동

  5. 5

    하단의 "변경사항 저장하기" 버튼을 클릭합니다.

✓ 체크리스트
✓
표시 이름을 샌드박스 실명으로 설정했다
